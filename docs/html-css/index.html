Medidas

Por defecto, los navegadores tenen font-size: 16px como tamaño de fuente porque equivalen aproximadamente a los 12 puntos de tamaño que se utiliza en imprenta (el cual es el tamaño al que el ojo humano está acostumbrado)

    Absolutas

  • 1px = 1/96 de 1in
  • 1cm = 96px/2,54
  • 1mm = 1/10 de 1cm
  • 1Q (cuarto de milimetro) = 1/40 de 1cm
  • 1in (inches) = 2,54cm = 96px
  • 1pc (picas) = 1/16 de 1in
  • 1pt (puntos) = 1/72 de 1in

    Relativas (dependen del contexto)

  • 1rem = 16px (corresponde a la medida de la "m"(en imprenta es la letra que mas ocupa) de la raiz del documento: html)
  • em = font-size: x (corresponde a la medida de la "m"(en imprenta es la letra que mas ocupa) del contexto donde nos encontramos)
  • % (respecto a al contenedor que lo contiene)
  • width:100% (al añadir margin, la caja seguira midendo el 100%) vs width:auto (al añadir margin, se reajustará para no salirse)
  • vw (viewport width)
  • vh (viewport height)
  • vmin
  • vmax
  • ex (altura x de la fuente del elemento)
  • ch (la medida del ancho del glifo "0" de la letra del elemento)
  • lh (altura de la linea del elemento)

Centrar contenido

Si queremos centrar un bloque, Se recomienda poner:
margin-left: auto
margin-right: auto

En lugar de:
margin: 0 auto
(porque de esta forma tocamos margenes verticales que no nos interesa tocar)

Centrar elementos

La propiedad text align:
Centra el contenido, NO el contendor de un elemento de blque
Se alinea RESPECTO AL ANCHO que le definimos

p#text-align{
    background-color: grey;
    border: 2px dashed black;
    padding: 20px;

    text-align: center; /*centra el CONTENIDO (no el contendor) */
    width: 500px; /* Se alinea respecto al ancho. Si no aplicamos un ancho cogerá el ancho completo */

    margin: auto; /* centra el CONTENDOR por que es un elemento bloque, auto intentará siempre ocupar el ancho completo*/
}

Centrando imagenes

img#imagen{ 
    background-color: gray;
    border: 2px dashed black;
    padding: 20px;

    /*💡 Las imagenes vienen con unas dimensiones de contenido establecidas por la imagen*/

    display: block; /* <- el elemento ocupa el ancho de la pantalla (pero como el contenedor tiene unas dimensiones establecidas, no abarcará VISUALMENTE toda la pantalla)

        width: 100px; <- para visualizarlo mejor, hay que imaginarse que la imagen tiene sus dimensiones puestas 
        height: 100px; 

    text-align: center; <- es inutil porque la ya se encuentra en el centro de su contendor 

    -------------------------------------*/

    margin-right: auto; /* <- lo centrariamos solo con el margen */
    margin-left: auto;  /* <- lo centrariamos solo con el margen */

    /* el margin auto se calcula respecto a la estructua de bloque, si no fuera display block
    no se podria calcular el auto por lo que se tendrían que poner valores en px en el margin */
}

Float

Práctica

Al definir un elemento como float, mantiene su posición, solo que ahora lo detecta unicamente los elementos inlines(independientemnte de lo anidados que esten) y otros floats, RECUERDA QUE NO PUEDEN CENTRARSE

Display

1º Contendor: BLOCK | Texto: INLINE
2º Solo lo detectan elementos INLINE
3º Si solo flotas este, veras que el div de abajo actuara como si no existiera el float, pero el texto si lo respetara hasta el punto de salirse
4º Solo lo detectan elementos INLINE
5º Contendor: BLOCK | Texto: INLINE
6º Contendor: INLINE-BLOCK | Texto: INLINE
7º Contendor: INLINE-BLOCK | Texto: INLINE
8º Contendor: BLOCK | Hijo: BLOCK
9º Contendor: BLOCK |Texto: INLINE (Vemos que afecta a cualquiera INLINE indpendientemente de lo anidado que este)
10º Contendor: INLINE| Texto: INLINE
11º Contendor: INLINE | Texto: INLINE
