La propiedad text align:
Centra el contenido, NO el contendor de un elemento de blque
Se alinea RESPECTO AL ANCHO que le definimos

p#text-align{
    background-color: grey;
    border: 2px dashed black;
    padding: 20px;

    text-align: center; /*centra el CONTENIDO (no el contendor) */
    width: 500px; /* Se alinea respecto al ancho. Si no aplicamos un ancho coger√° el ancho completo */

    margin: auto; /* centra el CONTENDOR por que es un elemento bloque, auto intentar√° siempre ocupar el ancho completo*/
}

Centrando con POSITION

.container{
    position: relative;
    height: 50px;

    border: 1px solid black;
}
.box{
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%,-50%);

    background-color: red;
    width: 30px;
    height: 30px;
}

Centrando con FLEXBOX

    display: flex;
    justify-content: center;
    align-items: center;
    height: 50px;

    border: 1px solid black;
}

    background-color: blue;
    width: 30px;
    height: 30px; 
}
